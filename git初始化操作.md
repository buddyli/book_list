1，初始化本地资源库
	git init
2，将文件添加到提交列表中
	git add <文件名>
3，提交文件到索引中
	git commit -m "注释内容"
4，将本地文件夹与资源库关联
	git remote add <资源库名称> “URL-这里最好使用ssh地址，https方式提交没成功”
5，真正提交文件-提交到主干
	git push -u <资源库名称> master
========================================
以上方式没有实验成功，本项目是先从githun创建一个资源库，将资源库同步到本目录
git pull <资源库地址>

然后再提交的。
